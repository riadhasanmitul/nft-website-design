# NFT Website Design

A modern and responsive **NFT landing page** built using **HTML, CSS, and JavaScript**. This project showcases a sleek user interface inspired by NFT marketplaces, focusing on responsive design, modern layouts, and engaging user experience.

> **Disclaimer:** This project is created for educational and portfolio purposes only. It is not affiliated with or endorsed by any NFT marketplace or blockchain platform.

---

## Features

- Modern NFT landing page
- Fully responsive design
- Hero section with call-to-action
- Featured NFT showcase
- Creator and collection sections
- Smooth scrolling and animations
- Clean and intuitive navigation
- Cross-browser compatibility
- Well-structured and maintainable code

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/riadhasanmitul/nft-website-design.git
```

### Navigate to the Project

```bash
cd nft-website-design
```

### Run the Project

Open `index.html` in your preferred web browser.

For a better development experience, use **VS Code Live Server**.
